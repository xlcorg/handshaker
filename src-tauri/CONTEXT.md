# IPC

Tauri-слой Handshaker: `#[tauri::command]`-команды, specta-биндинги, `AppState`
со сторами. Оборачивает ядро; ядро specta-free, поэтому проводные типы — это
`*Ipc`-зеркала в `src-tauri/src/ipc/`.

## Language

**Send**:
Единственная команда отправки запроса (`grpc_send`): принимает сырой черновик +
send ctx и владеет всей цепочкой «резолв-пайплайн → экспансия билтинов → invoke
→ инвалидация токен-кэша на UNAUTHENTICATED». Фронт шлёт шаблоны, не
разрешённые значения.
_Avoid_: grpc_invoke_oneshot (прежняя команда, принимавшая пред-разрешённый запрос)

**Send ctx**:
Ссылки на контекст резолва — `{collection_id?, env_name?}`. Именно ссылки:
коллекцию и окружение команда читает из своих сторов сама; фронт не собирает и
не пересылает их содержимое.
_Avoid_: оверлеи переменных (роль VarsResolveCtxIpc для редакторов, не для Send)

**Effective-auth запрос**:
Мелкая команда-вопрос «какой auth действует для этого черновика в этом ctx» —
ей UI (Auth-таб, снапшот истории) отвечает, не держа собственной копии правила
auth pick.
_Avoid_: дублирование env-гейта на фронте
