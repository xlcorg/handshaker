# Specs and plans are written in English

All documents under `docs/superpowers/specs/` and `docs/superpowers/plans/`
(including `archive/`) are written in English — headings, prose, status banners.

Code identifiers, file paths, and quoted UI strings stay verbatim regardless of
language. Conversation language does not matter: even when the discussion runs
in Russian, the written spec/plan is English.
